## Project Introduction

An AI intelligent image library platform based on Vue3 + Spring Boot.

### Features

- Supports user registration, login, personal information management, etc.
- Supports users to upload, manage, and share images.
- Supports image classification, tagging, and search functions.
- Administrators can review user-uploaded images to ensure content legality and quality.
- Supports opening spaces, users can purchase storage space to upload more images.
- Supports image download and sharing functions, with real-time collaborative editing of images.

### Technology Stack

- Frontend: Vue3, Ant Design, Axios, Pinia
- Backend: Spring Boot, MyBatis-Plus, Redis, Caffeine, Jsoup
